# Converter

> T9 conversion tool

Installation
----

+ Follow installation steps of the [backend](https://github.com/KarinaDavtyan/converter-backend)

+ Run commands :arrow_down:

`git clone https://github.com/KarinaDavtyan/converter.git`

`cd converter`

`yarn install`

`yarn start`

or

`yarn test` for testing
